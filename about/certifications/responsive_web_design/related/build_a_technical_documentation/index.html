Introduction

Welcome to the technical documentation page for HTML programming.

HTML stands for HyperText Markup Language and is a computer language used to write code for internet websites. HTML is almost always used alongside Cascading Style Sheets, also known as CSS, which defines the visual appearance of a website.

Basic Structure

HTML is similar to other markup languages in that components are defined through the use of opening and closing tags formed with angle brackets. For example, the opening and closing tag for a div element would look like the following, respectively:

<div> </div>

Every HTML file should begin with the following line of code. This line explicitly tells the browser that this file is HTML code.

These are the three primary elements that compose the HTML website structure:

  • html
  • head
  • body

The basic structure of an empty HTML page would look like this:

<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
  </body>
</html>

Headings

HTML contains multiple heading elements that can be used to display 'heading' text on a site. The largest heading type should only be used once to display the topic of the page. Subsequent headings can be used to lead different sections of the content.

The biggest heading in HTML is the h1 element. Headings are then used in descending order by counting upwards: h2, h3, and so on.

This would create a large title text of "Hello World!":

<h1>Hello World!</h1>

The style of text can be altered by wrapping text in certain elements. For example, text can be boldened, italicized, crossed through, and more. Here are some elements that can be used to decorate text:

  • strong (boldens)
  • em (emphasizes)
  • s (strike through)
  • ins (underline)

Images

The last topic that is covered in this documentation is images. Images can be inserted into pages to display image files such as .png, .jpg, or .webp. To do so, create an img element. Unlike most other elements, images do not use closing tags. A src attribute should be set with the image path or image URL using the format "attribute=value".

<img src="./dog.png">
